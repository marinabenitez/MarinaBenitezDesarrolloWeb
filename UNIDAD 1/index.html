Generalidades del Curso

Fundamentación

Objetivos

Al finalizar el curso se espera que las y los estudiantes puedan:

  • Crear páginas web utilizando HTML, CSS, SVG y JavaScript.
  • Escribir el 100% del código de un sitio web.
  • Diseñar una web que se adapte a diferentes tamaños de pantallas y gestionar la navegación.
  • Crear elementos animados que interactúen con el usuario.
  • Consumir API´s con JavaScript para colocar contenidos en un sitio.

Contenido

UNIDAD 1: HTML y Desarrollo Web

Entorno de desarrollo. Editor de código. Inspector y herramientas de desarrollo de los navegadores. Funcionamiento básico de un sitio web. Elementos de un sitio web. Etiquetas y atributos. Etiquetas de texto. Imágenes y enlaces. Etiquetas de bloque. Etiquetas de documentos. Repositorios y Git.

UNIDAD 2: Introducción a la CSS

Vincular un archivo CSS. Sintaxis básica de CSS. Selectores. Estilos y tipografías. Estructura en las hojas de estilo. Selectores avanzados. Estilos de navegación. Rests CSS. El modelo de la cada. Displays. Floats. Positions. Estilos para listas. Border, border-radius y box-shadow. Fondos e imágenes de fondo. WebFonts. Colores y opacidad. Bordes y redondeados. Sombras. Degradados y fondos.
